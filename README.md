# Votación VDI

Aplicación para gestionar procesos de votación de forma simple, transparente y trazable.  
Este proyecto está orientado a centralizar la creación de elecciones, el registro de participantes y el conteo de resultados en una sola plataforma.

## Descripción

**Votación VDI** busca resolver el proceso de votación digital con enfoque en:

- Claridad para usuarios y administradores.
- Integridad de los datos de votación.
- Resultados rápidos y auditables.
- Facilidad de despliegue y mantenimiento.

## Ideas principales

- **Elecciones configurables:** creación de eventos de votación con opciones personalizadas.
- **Gestión de votantes:** control de acceso y validación de participación.
- **Emisión de voto:** flujo guiado para votar una única vez por proceso.
- **Conteo automático:** cálculo de resultados en tiempo real o al cierre.
- **Trazabilidad:** registro de eventos relevantes para auditoría.
- **Panel administrativo:** herramientas para abrir/cerrar votaciones y revisar métricas.

## Objetivos del proyecto

1. Reducir errores manuales en procesos de votación.
2. Mejorar la transparencia en la publicación de resultados.
3. Ofrecer una base escalable para futuras mejoras (reportes, roles, seguridad avanzada).

## Alcance funcional (MVP)

- Crear una votación.
- Registrar opciones/candidatos.
- Habilitar votantes.
- Emitir voto.
- Mostrar resultados consolidados.

## Cómo ejecutar la aplicación

1. Clona el repositorio:
  ```bash
  git clone "link del proyecto"
  cd votacion-vdi
  ```
2. Instala las dependencias:
  ```bash
  npm install
  ```
3. Inicia la aplicación:
  ```bash
  npm start
  ```

La aplicación estará disponible en `http://localhost:3000` por defecto.
